# 组织情报信息

Public GitHub Pages mirror for organization intelligence reports.

This repository contains only share-safe, sanitized files:

- public daily reports
- public stage reports
- public research framework

It intentionally excludes private source-channel indexes, local reference documents,
raw archive materials, desktop paths, credentials, and unverified private materials.

## SuperClaude Boundary

Use the local `SUPERCLAUDE_ORCHESTRATION_SKILL.md` before publishing sanitized reports.

Required checks:

- `/sc:research`: confirm claims are source-backed and evidence levels are explicit.
- `/sc:analyze`: confirm private channels, internal links, local paths, screenshots, credentials, and personal information are absent.
- `/sc:document`: update public framing only with share-safe methodology, evidence levels, and source indexes.
