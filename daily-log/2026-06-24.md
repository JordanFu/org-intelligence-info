## 传统大厂与 AI 原生组织分组观察

- **传统大厂：** 今天没有发现新的公司级绩效、晋升、职级、干部或组织结构调整硬证据。中文命中多围绕腾讯 WXG 项目负责制、阿里 Token Foundry、百度职级并轨继续解释，但时间锚分别落在 2026 年 5 月、6 月上旬或更早。
- **AI 原生组织：** 今天的有效信号仍集中在岗位族群，而不是制度公告。DeepSeek Harness 显示国内模型公司也在把 Agent 产品化拆成研究、工程、产品角色；海外公司则继续通过 FDE、Applied AI、Post-training、Safety/Sovereign AI 岗位面显性化组织边界。
- **趋同处：** 国内外 AI 原生公司都在处理同一个组织问题：模型能力如何进入真实工作流，并在研究、工程、产品、客户现场之间建立翻译层。
- **分化处：** 海外公司更常通过官方 careers / Ashby / Lever 岗位页暴露组织边界；国内公司今天更依赖负责人社交招聘和媒体转述，证据硬度较低。

## 组织情报结论卡

### 结论卡 A｜DeepSeek Harness 是 Agent 产品化组织边界的 L2 观察

- L1 事实：多源报道显示 DeepSeek Harness 团队开放 Harness 研究员、工程师、产品经理三类岗位，并由负责人公开招聘。
- L2 机制判断：Agent 能力产品化正在逼出比“模型研究”更细的工作单元，至少需要研究前沿定义、工程架构实现和产品形态定义三类接口。
- L3 组织判断：暂不成立。当前证据不足以证明公司级组织重构、事业部化或成熟管理体系。
- 证据置信度：中。
- 阅读价值：持续观察。
- 适用边界：适合观察模型公司如何从模型能力进入 Agent 产品化，不适合推断绩效、职级、汇报线或管理机制变化。
- 待验证问题：官方岗位页、团队归属、管理层和发布节奏。
- 入库等级：L2。

### 结论卡 B｜海外 FDE / Applied / Deployment 岗位面继续形成 AI 原生组织基线

- L1 事实：OpenAI、Mistral、Perplexity、Cohere 等公司继续出现 FDE、Applied AI、AI Deployment Engineer、Sovereign AI 等岗位面。
- L2 机制判断：研究突破到企业采用之间正在形成前线技术组织，承担客户现场、产品反馈、工程集成和部署质量接口。
- L3 组织判断：作为长期基线成立；但 2026-06-24 未发现新的 L3 拐点。
- 证据置信度：中高。
- 阅读价值：背景参考 / 持续观察。
- 适用边界：复杂企业部署、公共部门、私有化和高价值工作流；不适合泛化到所有 AI 产品。
- 待验证问题：是否出现 manager 层、区域分线、客户分层和交付质量机制。
- 入库等级：L2-L3 基线延续。

### 结论卡 C｜今日传统大厂扫描的主要价值是时间窗降噪

- L1 事实：腾讯项目负责制、阿里 Token Foundry、百度职级并轨等命中均可回溯到 2026 年 5 月或 6 月上旬。
- L2 机制判断：传统大厂 AI 组织调整仍有机制价值，但今天公开材料没有提供新的权责、资源、评价、岗位或流程变化。
- L3 组织判断：今天不新增传统大厂组织机制结论。
- 证据置信度：高。
- 阅读价值：暂不采纳为新增，作为反证记录。
- 适用边界：适合维护历史基线和避免误判，不适合作为今日趋势判断。
- 待验证问题：是否后续出现官方会议、财报、事业群调整或更多部门互证。
- 入库等级：反证/背景。
